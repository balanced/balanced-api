## Endpoints

###ACH Debit

    ach_debits
    - id
    - bank_account
    - amount
    - available
    - rejected


### Bank Account

    bank_accounts
    - id
    - name
    - account_number
    - routing_number
    - type
    

## Examples


`POST /ach_debits`

    {
        "bank_account": {
            "name": "Gottfried Leibniz",
            "account_number": "3819372930",
            "routing_number": "121042882",

    }
